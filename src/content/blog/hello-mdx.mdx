export const buildYear = new Date().getFullYear();

## MDX とは

MDX は Markdown の拡張で、記事の中に JavaScript の式やコンポーネントを埋め込める。

たとえばこの行の西暦は式で出力している: **{buildYear} 年**

## 使い分け

- 通常の記事 → `.md` で十分
- 動的な要素やコンポーネントを使いたい記事 → `.mdx`

どちらも同じ `src/content/blog/` フォルダに置くだけで記事として認識される。
