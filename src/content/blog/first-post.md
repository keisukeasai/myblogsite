## ブログを始めた

Astro と Tailwind CSS を使ってこのブログを構築した。この記事は Markdown で書かれている。

### Astro を選んだ理由

- **表示速度が速い** — デフォルトで JavaScript をほとんど出力しない
- **Markdown / MDX を標準サポート** — 記事をファイルとして管理できる
- **TypeScript 対応** — フロントマターにスキーマを定義できる

### コードも書ける

```ts
const greeting: string = "Hello, Astro!";
console.log(greeting);
```

今後もここに記事を追加していく。
