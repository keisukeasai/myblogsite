# Brand Guidelines — かえりたくなる部屋

ブランドの一貫性を保つための指針。UI・記事・画像・コピーはすべてこの世界観に合わせる。

---

## 1. ブランドアイデンティティ

| 項目 | 内容 |
|---|---|
| ブランド名 | かえりたくなる部屋 |
| メッセージ | **帰りたくなる毎日を。** |
| 説明 | ガジェットとインテリアで、毎日の暮らしを少し心地よく。 |
| 位置づけ | 商品を売る場所ではなく、**暮らしが少し豊かになる体験**を届けるライフスタイルメディア |
| 目指す世界観 | Apple / MUJI / BALMUDA / Nothing — **余白・静けさ・上質さ** |

---

## 2. カラーパレット

基軸はロゴの3色: paper `#FAF9F5` / ink `#222` / sage `#A7B8A2`。
実体は [`src/styles/global.css`](../src/styles/global.css) の `@theme`。**派手な原色・高彩度の青は使わない。**

### アクセント（sage / ロゴの葉色）
唯一のブランドアクセント。リンク・強調・アクティブ状態に限定して使う。

| Token | Hex | 用途 |
|---|---|---|
| `accent-50` | `#f4f6f2` | 淡い背景・コンセプト帯 |
| `accent-100` | `#e6ece2` | ホバー背景 |
| `accent-200` | `#ccd8c6` | 選択ハイライト |

| `accent-600` | `#53644b` | ラベル文字（ライト） |
| `accent-700` | `#43503d` | リンク文字（ライト） |

### ニュートラル（warm / light）

## 3. タイポグラフィ

| 用途 | フォント | 指定 |
|---|---|---|
| 本文・見出し（日本語） | Noto Sans JP Variable | `font-sans` |
| 装飾ラベル（英字・大文字） | Montserrat Variable | `font-display` |

- **英字ラベル**（例: `GADGET & INTERIOR MEDIA`）は `font-display` + `uppercase` + `tracking-[0.2em]` + sage。セクションの目印として小さく使う。
- **見出し**は `font-bold`、日本語の行間は広め（見出し `leading-relaxed`〜`snug`、本文 `leading-loose`）。
- **italic は原則使わない**（静けさを損なうため。旧デザインの装飾italicは廃止済み）。

---

## 4. ロゴ / ファビコン

- ロゴは [`src/components/Logo.astro`](../src/components/Logo.astro)。アイコンと文字は `currentColor`で描画され、親の文字色に追従（ライトで ink、ダークで cream）。
- **葉のグリーン `#A7B8A2` は固定**（両テーマ共通のブランドカラー）。
- ファビコンは [`public/favicon.svg`](../public/favicon.svg)。`prefers-color-scheme` でライト/ダーク自動切替。
- ロゴ周囲には十分な余白を確保し、背景色の上に重ねる際もコントラストを保つ。

---

## 5. 形・影・余白

- **角丸**: カード `rounded-2xl`、大きな面 `rounded-3xl`、チップ `rounded-full`。
- **影**: 既定より弱い暖色影（`shadow-sm`〜`shadow-lg`）。**強い影・濃い影は使わない。**
- **余白最優先**: セクション間は広く（`mt-16`〜`mt-24`）。詰め込まない。本文幅は `max-w-reading`（約44rem）。

---

## 6. トーン & ボイス

- **静か**: 誇張・煽り表現を避ける。「最強」「神」より「心地よい」「ちょうどいい」。
- **正直**: 良い点だけでなく、気になる点・向き不向きも書く。
- **暮らし目線**: スペックの羅列でなく、毎日の体験を起点に語る。
- 文体は丁寧語ベース。読者を急かさない。

---

## 7. Do / Don't

| ✅ Do | ❌ Don't |
|---|---|
| paper / warm neutral を基調に余白を活かす | 高彩度の青・原色・派手なグラデーション |
| アクセントは sage を一点投入 | 複数の強い色を同時に使う |
| 静かな装飾・弱い影 | 濃い影・過度な装飾・italic見出し |
| 正直で落ち着いたコピー | 煽り・誇張・記号の多用 |
